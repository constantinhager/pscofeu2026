# Pester - deep dive into Mocking.

## Abstract

Are you ready for some good old hardcore geeking out? This session will show you the most complex part of Pester - the Mock command, and all the tricks that we use there.

I will explain how scopes work in PowerShell and how we jump between them. How we use variable shadowing, and how we communicate between Pester and your function, without publishing any Pester internals. You will see what the mock prototype is, why it is full of ugly code and why it has such a long name.

This session will teach you the PowerShell trivia that turns you into PowerShell wizard!
