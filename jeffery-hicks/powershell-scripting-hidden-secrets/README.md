# PowerShell Scripting Hidden Secrets

## Abstract

We all love PowerShell because it makes our work easier and is even fun to use. However, PowerShell is big and it is impossible to know everything. That's why we have this event, and you should attend this session on 10 hidden PowerShell scripting secrets. These are concepts and techniques that aren't well documented or are little used. This demo rich session will be led by veteran PowerShell expert Jeff Hicks.
