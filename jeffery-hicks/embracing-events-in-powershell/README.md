# Embracing Events in PowerShell

## Abstract

It is a fact of life working in IT that things happen. Something is always being created, modified, or removed. Most of the time we let these things happen without a second thought. But there may be situations where is something happens, you want to know. Or more specifically, you want to do something! In this full of real-world examples, you'll learn how to take advantage of event management in PowerShell. Perhaps you want to run cleanup tasks automatically when you close a PowerShell session. Or need to know when the contents of a folder or file changes. These are a few of the situations we'll explore in the code-heavy session. Led by long-time PowerShell author and teacher Jeff Hicks.
