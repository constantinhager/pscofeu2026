# Microsoft Graph with PowerShell 101

## Abstract

In this follow-along, we will explore Graph automation by creating a Microsoft Entra app registration, an Azure Automation runbook to check your other Entra apps for expired certificate or credential secrets. All purely PowerShell, of course.

This session is ideal for people new to Entra and Graph, but also for intermediate users. We will explore the differences between Enterprise Apps and App Registrations, Delegated and Application permissions as well as authentication flows.

To make the most of this follow-along, be sure to create your own, free Entra tenant to try out all that stuff. If you cannot create a tenant, credentials with a very minimal set of permissions will be given to you, and your experience will be a little less interesting.
