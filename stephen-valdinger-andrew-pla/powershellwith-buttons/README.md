# PowerShell…..with buttons

## Abstract

Every PowerShell developer can be a frontend developer with a little help from PowerShell Universal. Learn how to build tools that harness your existing automation, add API capabilities, or turn a script into an app that anyone in your organization can use.

This session takes a long look at PowerShell Universal and helps you understand how to leverage the platform to transform the way you design tools in your organization. You’ll learn about Automation, Web applications, and even how to generate REST API endpoints for your scripts. 

We’ll also look at how you can leverage AI to make things look pretty, because let’s face it….we’re PowerShell developers, not CSS wizards.
