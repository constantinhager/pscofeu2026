# Authoring and using DSC v3 resources

## Abstract

DSC v3 empowers users to author new resources in any language and continue using existing ones through adapter support. To illustrate the flexibility and power of DSC v3, this session introduces the cross‑platform sshd_config resource, which is built to manage OpenSSH server configuration safely and consistently. This resource highlights the kinds of real‑world configuration challenges DSC v3 is designed to address, especially in security‑critical environments. Using the creation of the sshd_config resource as a case study, this session will focus on best practices for designing, validating, and consuming DSC v3 resources.
