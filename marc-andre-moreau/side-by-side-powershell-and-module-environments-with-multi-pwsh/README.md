# Side-by-Side PowerShell and Module Environments with multi-pwsh

## Abstract

Testing scripts across PowerShell versions and managing module dependencies can quickly turn into a nightmare. In this hands-on demo, we’ll use multi-pwsh to install and run multiple PowerShell versions side-by-side, while creating isolated module environments similar to Python virtual environments. You’ll see how to launch version-specific shells, keep module dependencies separated, and export environments for reuse across systems.
