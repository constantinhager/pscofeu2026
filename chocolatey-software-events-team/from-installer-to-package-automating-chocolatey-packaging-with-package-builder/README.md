# From Installer to Package: Automating Chocolatey Packaging with Package Builder

## Abstract

Packaging software for enterprise deployment often involves repetitive scripting, manual testing, and ongoing maintenance. Chocolatey's Package Builder helps automate much of this work by generating Chocolatey packages directly from installers.

In this 15-minute session, we’ll walk through how Package Builder accelerates package creation, helps standardize packaging practices, and reduces the effort required to maintain internal software repositories. Whether you’re building your first internal packages or looking to streamline an existing workflow, you’ll see how automation can make packaging faster and more consistent.
