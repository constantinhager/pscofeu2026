## Jan Egil Ring

<img src="https://sessionize.com/image/009b-200o200o2-Hjw2D8QoFQMt3SJsxFdNm4.jpg" alt="Jan Egil Ring" style="max-width:300px; max-height:300px; width:auto; height:auto;" />

*Sr Cloud Solution Architect, Microsoft*

Jan Egil works as a Sr Cloud Solution Architect at Microsoft - helping customers migrate, transform and modernize into Azure. At Microsoft, he is also a Core Maintainer on the Azure Arc Jumpstart team.

He has a wide experience with automation and a strong passion for PowerShell.

Before joining Microsoft, he was a multiple-year recipient (2011-2022) of the Microsoft Most Valuable Professional Award for his contributions to the Windows PowerShell, Cloud & Datacenter Management, and Microsoft Azure technical communities.

He speaks regularly at user groups and conferences, such as Nordic Infrastructure Conference (NIC), PowerShell Conference Europe, and PowerShell Summit.
