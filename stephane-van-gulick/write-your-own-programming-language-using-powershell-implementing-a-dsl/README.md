# Write your own programming language using PowerShell: Implementing a DSL

## Abstract

Write your own programming language using PowerShell by implementing a Domain Specific Language (DSL).

In this session, we’ll explore the core principles of DSL design. And show you all the steps involved into creating your own DSL.

We’ll start with the basics by explaining what a DSL is and showcasing some well-known examples.

Next, we’ll dive into the fundamental building blocks of creating a DSL, breaking down the theory behind the DSL design, and which theoretical building blocks exists, and in which order to use them. 

We’ll walk through the process of designing a DSL, AND writing our own DSL.
Finally, we will walk through real-world DSL's such as Pester, PsGraph or Pshtml and look at how community has implemented the DSL Pattern in these cases.

By the end of this session, you’ll have the knowledge and inspiration to design your own PowerShell DSL.
