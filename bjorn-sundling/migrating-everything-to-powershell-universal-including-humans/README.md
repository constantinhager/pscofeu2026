# Migrating everything to PowerShell Universal - Including humans!

## Abstract

"We have lots of home made old tools built in various languages and no one knows how they work or how we update them."
A tale as old as time.

PowerShell Universal in an amazing tool to build apps, APIs, and much much more using PowerShell alone. Join me in a journey showing not only what PowerShell Universal can do, but how it helped an IT-department move from a mix of VB.Net, C#, and ancient homebuilt PowerShell GUI tools to a unified way of working, and in the process managed to get all of IT involved in developing and inventing new and (hopefully) future-proof tools using PowerShell.

This is the story about how we changed "old" to "new", Various" to "standardised", and "no one" to "everyone".
