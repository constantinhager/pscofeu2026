# Argument completers, Dynamic parameters, and more: Building better PowerShell functions

## Abstract

Wouldn't it be great if there was a way to inject a level of control over how people interacted with the tools you provide them beyond "well thought out parameters"? Join me in this session, where we will explore some of the more advanced features of PowerShell functions that so often go unmentioned, such as argument completers, dynamic parameters, etc., and how we can leverage these advanced features to add control and validation to your PowerShell toolbelt!
