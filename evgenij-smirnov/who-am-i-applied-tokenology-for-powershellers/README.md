# Who Am I? Applied Tokenology for PowerShellers

## Abstract

Much has been said about Web Tokens and all the different ways of forging, stealing and abusing them. But what about the security tokens that have been there even before "the cloud" became a thing?
In this session, we will look at different ways for your PowerShell code to determine what security context it is running in, what privileges it can expect on different systems, and how to adjust these if needed (and allowed).
We will concentrate mostly on Windows with a sidelong glance cast on Linux as well.
