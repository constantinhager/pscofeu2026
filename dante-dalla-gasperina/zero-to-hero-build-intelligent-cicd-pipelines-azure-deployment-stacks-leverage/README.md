# Zero-to-Hero: Build Intelligent CI/CD Pipelines Azure Deployment Stacks Leverage

## Abstract

Learn how to setup GitHub actions or ADO tasks for Continues Deployment Stack CRUD while adding validation steps along the way provided by Stacks What-if. In this session we will follow along with the latest technologies provided by Azure Deployments team, ranging from the latest Bicep language features, template preview validation from What-if, and fast environment cleanup via Deployment Stack updates.
